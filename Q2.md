# 📅 Day 1–3: JavaScript Core (Deep Thinking)

## 🔹 Section 1: Output-Based Questions

### Q1

```js
console.log([] + []);
console.log([] + {});
console.log({} + []);
```

👉 Explain WHY output is coming.

---

### Q2

```js
console.log(1 + "2" + 3);
console.log(1 + 2 + "3");
```

👉 What is type coercion here?

---

### Q3

```js
for (var i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 1000);
}
```

👉 Why same output?
👉 Fix it.

---

### Q4

```js
let obj = {
  name: "JS",
  getName: function() {
    return this.name;
  }
};

let fn = obj.getName;
console.log(fn());
```

👉 Why undefined?
